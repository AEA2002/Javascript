Events

Attach event listeners
* HTML:        onclick="function()"
* Traditional: thing.onclick=function;
* Event listeners: thing.addEventListener('click', function, t/f {capture});

The event object (e)

All events have target, type, cancelable properties and preventDefault(), stopPropagation() methods

target is important because you can attach an event listener to the parent element and inspect the event's target.
